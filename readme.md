# Talk to Gimini

is a multimodal interface for talking to gimini via api.

## install
python 3.12 needed

python -m venv .venv
source .venv/bin/activate 
pip install --upgrade pip 
pip install -r requirements.txt

## setup .env
GEMINI_API_KEY=<key>
from https://aistudio.google.com/app/apikey?hl=de
